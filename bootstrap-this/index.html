About Bootstrap

Open Source

Bootstrap is an open source for developers to use when creating digital content for the web. Bootstrap's functionality works with the languages HTML, CSS and JS. Bootstrap can allow you to quickly protoype your ideas and bring them into reality. It is built on a responsive grid system with extensive prebuilt components.

Mobile Firts

Bootstrap is built on a mobile-first mockup design. With the responsive twelve column system allows you to easily create a responsive website that can be viewed on any device. Bootstrap includes many variables and mixins from SASS that provide you with many pre-determined classes

Components Library

Bootstrap comes with many different components that allow you to create any specific components. Need a contact form? Bootstraps got it!, Need a drop down menu? Bootstraps got it! With no knowledge of JavaScript you can create anything to your hearts content.

Components

Alerts

Provide feedback from user actions allowing them to know if they are going to click on something dangerous, or something that will allow them to go somewhere else.

Breadcrumbs

Indicate to the user where they are on a webpage with a visual hierarchy to avoid them getting either confused or lost on a webpage

List Groups

List groups display a series of cotent that the user can go through and look at to further user engagement throughout the webpage.

Cards

A card is what the user is viewing now. It is a flexible and extendible container that can either display images and text or just text with or without background color.
